## Multi-Agent LangGraph System

A multi-agent system built with LangGraph that demonstrates orchestration between specialized agents using either OpenAI or Ollama language models.

## First time setup:
```bash
# Create and activate virtual environment
python -m venv venv
./venv/Scripts/activate

# Install dependencies
pip install -r requirements.txt
```

## Regular startup:
```bash
# Activate virtual environment
./venv/Scripts/activate

# Run the application
python ./src/main.py
```
